# 0x05. Python - Exceptions

This project focuses on handling errors and exceptions in Python using `try`, `except`, `else`, and `finally`. It teaches how to write robust code that avoids crashes when encountering unexpected runtime data or edge cases.

## Learning Objectives
* Why programmers must be prepared for errors.
* What are exceptions and how to use them.
* When to execute a code block only if no exception occurs (`else`).
* When to execute a cleanup code block regardless of exceptions (`finally`).
* How to raise built-in exceptions manually.
