=================================
How to use MyList
=================================

This module defines a `MyList` class that inherits from the built-in `list`.

Test basic usage:
>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

Test with negative numbers:
>>> my_list2 = MyList([-5, 10, 0, -2])
>>> my_list2.print_sorted()
[-5, -2, 0, 10]

Test with empty list:
>>> my_list3 = MyList()
>>> my_list3.print_sorted()
[]

Test with one element:
>>> my_list4 = MyList([5])
>>> my_list4.print_sorted()
[5]
