# 0x03. Python - Data Structures: Lists, Tuples

This project explores Python data structures, specifically **Lists** and **Tuples**. We cover how to access, modify, and iterate through them, as well as how to use built-in methods to manipulate data effectively.

## Learning Objectives
* How to use lists and tuples.
* How to iterate through lists.
* How to use list methods (append, pop, insert, etc.).
* Understanding the difference between lists and tuples.
