# Python - If/else, loops, functions

## Description
This project is part of the higher-level programming curriculum. It covers the foundational concepts of Python programming, specifically focusing on control flow statements (`if`, `elif`, `else`), loops (`for`, `while`), and the basics of defining and using functions.

## Learning Objectives
* Why Python programming is awesome
* How to use conditional statements (`if`, `elif`, `else`)
* How to use loops (`for` and `while`)
* The purpose and usage of `break` and `continue` statements
* How to use the `range` function
* What are functions and how to use them

## Technologies
* Language: Python 3.4+
* Operating System: Ubuntu 20.04 LTS
* Style Guide: `pycodestyle`

## Files
* **0-positive_or_negative.py**: Python script that checks whether a randomly generated signed number is positive, negative, or zero.
