# 0x04. Python - More Data Structures: Set, Dictionary

This project focuses on advanced data structures in Python, specifically **Sets** and **Dictionaries**. It covers their operations, efficiency, and functional programming tools like `map`, `filter`, and `reduce`.

## Learning Objectives
* What are sets and how to use them.
* What are the most common methods of set and how to use them.
* When to use sets versus lists.
* What are dictionaries and how to use them.
* When to use dictionaries versus lists or sets.
* What is a key and how to iterate over a dictionary.
* What are `map`, `filter`, and `reduce` functions.
